Decodificador de Textos

Aplicação para codificar e decodificar textos

Este projeto foi desenvolvido durante a formação Programador Iniciante da Turma 4 do Oracle Next Education. Trata-se de uma aplicação cujo objetivo é codificar e decodificar textos seguindo uma lógica pré estabelecida.

Desenvolvi o layout do projeto baseando-me em um modelo do Figma fornecido durante o curso, porém também coloquei no layout meu toque pessoal. Durante o desenvolvimento do layout apliquei o conceito de mobile first. A aplicação ficou responsiva tanto para smartphones, como tablets e desktop. Também adicionei ao projeto a opção de dark mode.

Tecnologias utilizadas: HTML5 | CSS3 | Javascript

Projeto Store API

API para gerenciamento de uma loja

Projeto back-end desenvolvido durante o Bootcamp de NodeJs da XP Educação. O projeto conta com uma banco de dados relacional construído com o PostgreSQL, para cadastro e gerenciamento de produtos, clientes e fornecedores. Também foi incluído um banco de dados noSQL, criado com o MongoDB, para armazenar a descrição dos produtos.

Na construção da API foram observados os princípios do SOLID e foram utilizados corretamente os métodos HTTP para requisições e respostas.

Tecnologias utilizadas:NodeJs | Javascript | Express | Sequelize | Mongoose | PostgreSQL | MongoDB

Gerenciador de Tarefas

Aplicação estilo to-do-list

Projeto Full-Stack de uma poderosa aplicação para cadastro e gerenciamento de tarefas do dia a dia. A aplicação conta com uma APIRest desenvolvida em NodeJs. Para utilizar a aplicação o usuário deve cadastrar um login e senha. Após realizar a autenticação de login o usuário terá acesso ao painel de tarefas, onde poderá cadastrar, atualizar, deletar e filtrar suas tarefas.

O front-end da aplicação foi desenvolvido em ReactJs, seguindo conceitos de mobile first e princípios de UX/UI. O repositório da API e do front se encontram em meu Github.

Tecnologias utilizadas: NodeJs | ReactJs | Javascript | Express | JWT | md5 | Bootstrap | Sass | Mongoose | MongoDB

Microsserviço de Notificações

Projeto Back-end em Typescript

Desenvolvimento de uma API para um serviço de notificações, realizado durante as aulas do Ignite Lab promovido pela Rocketseat. A API foi desenvolvida com o NestJs, utilizando a linguagem Typescript e o SQLite para a criação de um banco de dados relacional.

Durante o desenvolvimento da API também foi utilizado o Jest para elaboração de testes de integração automatizados. A API foi desenvolvida seguindo os princípios do SOLID.

Tecnologias utilizadas: NestJs | Typescript | Jest | Prisma | SQLite
