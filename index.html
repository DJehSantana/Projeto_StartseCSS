Projeto 1

Aprendendo Lógica de Programação

Neste projeto desenvolvemos vários algoritmos e fluxogramas baseados em situaçãoes do dia a dia. Desde uma receita de bolo até um programa simples para calcular a média dos alunos, aprendemos que a lógica está em tudo o que fazemos e em como solucionamos os problemas do dia a dia!

Projeto 2

Banco de dados SQL

Este projeto foi criado pensando em uma loja que vende doces. O nome da nossa loja é Candies. No banco de dados de nossa loja temos registrados nossos clientes e fornecedores. Também temos um plano de fidelidade que atribui pontos aos clientes de acordo as compras que realizam na loja. Nossas compras e vendas ficam muito bem registradas e organizadas em nosso Banco de Dados relacional!

Projeto 3

Aprendendo Html

Nesse projeto, desenvolvemos um mini site, explorando as tags básicas do HTML. Aprendemos a importância das tags metas e das tags semânticas. Também exercitamos boas práticas de organização de código!

Projeto 4

Estilizando com CSS

Aproveitando o código HTML desenvolvido no projeto anterior, trabalhamos a estilização do nosso mini site, usando o CSS. Nosso mini site finalmente ganhou vida e cor! Aprendemos a centralizar objetos e textos. Também descobrimos muitas coisas legais que podemos fazer com o CSS, como as animações que você viu nesse site!
